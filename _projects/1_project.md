GNN-Suite is a flexible benchmarking framework designed to systematically evaluate graph neural network architectures on biomedical informatics tasks. The framework provides standardised datasets, evaluation protocols, and modular GNN implementations to enable fair and reproducible comparisons across different architectures and biomedical applications.

**Key contributions:**
- Standardised benchmarking protocol for GNNs in the biomedical domain
- Modular framework supporting multiple GNN architectures (GCN, GAT, GraphSAGE, etc.)
- Comprehensive evaluation across diverse biomedical datasets
