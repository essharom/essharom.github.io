A full-day tutorial at **ISMB 2024** in Montreal, co-presented with I. Simpson, B. Ryan, C. Lee, and H. Issa.

This tutorial introduced participants to network analysis from first principles using real-world multi-modal data from the Generation Scotland study. Participants performed hands-on end-to-end network construction and computational analysis, gaining skills to develop their own network analytic pipelines.

**Topics covered:**
- Core network concepts and fundamentals
- Network construction for uni-modal and multi-modal data (e.g. gene expression + DNA methylation)
- Functional annotation analysis and community clustering
- Graph neural network-based classification
