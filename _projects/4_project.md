A collaboration with the LINK-Group at Semmelweis University investigating how translocating proteins compartment-specifically alter the fate of epithelial-mesenchymal transition (EMT) using compartmentalized Boolean network models.

**Key contributions:**
- Analysed and visualised simulation output of EMT using Python (seaborn, pandas)
- Contributed to understanding how protein translocation affects cancer cell state transitions
